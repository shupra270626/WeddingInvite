# Wedding Invite

This is a static wedding invitation site for Shubham and Pranjali.

## Deploy

No build step is required.

Upload these files and folders to any static web server while keeping the same structure:

- `index.html`
- `styles.css`
- `script.js`
- `assets/`

It will work on common static hosts such as Netlify, Vercel static output, GitHub Pages, Firebase Hosting, or a basic Nginx/Apache server.

## Photo Swap Points

- Hero background photo: `assets/img/US.jpeg`
- Bride photo placeholder: `assets/pranjali-placeholder.svg`
- Groom photo placeholder: `assets/shubham-placeholder.svg`

When you get the individual photos later, replace the two placeholder files in `index.html` with the new relative paths.
